# agGridAdapter | Дизайн-система Consta

Адптер для библиотеки [ag-grid-react](https://www.ag-grid.com/react-data-grid/getting-started/). Стилизует таблицу для [дизайн-системы Consta](https://consta.gazprom-neft.ru/).

[Посмотреть на GitHub](https://github.com/consta-design-system/ag-grid-adapter)

## Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/ag-grid-adapter` или `npm install @consta/uikit @consta/ag-grid-adapter`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/ag-grid-adapter` были вложены в компонент `Theme`.

[Как подключить тему](http://uikit.consta.design/?path=/docs/thematization-how-to-connect-theme--page)
